# minari-bot

## 주요기능
- **전략구독** : Binance / Upbit 거래소에 대한 유저 계정의 거래 API를 등록하고, 미나리봇에서 제공하는 전략을 구독
- **카피트레이딩** : 연결된 API로 서버에서 Tradingview 매수 시그널을 통해 자동매매를 실행
- **실시간 모니터링** : 거래소를 24시간 모니터링, 자동매매를 실행
- **대시보드** : 미나리봇을 통해 거래한 내역, 자산, 수익률을 기간별로 확인
